# file_upload
upload files with node.js
### initial
```
npm init
npm install express ejs multer --save
...
```
### how to use
```
git clone git@github.com:scott-x/file_upload.git
cd file_upload
npm install
npm run start
```
### Open yur browser and search `http://127.0.0.1:3000/`
